Astro ships with built-in syntax highlighting, so Compass can render fenced code blocks in Markdown and MDX without any extra setup. In this theme, code samples use Shiki with the `github-light` theme for a clean documentation-friendly look.

## Add a code block

To highlight a snippet, write a fenced code block and add the language name after the opening backticks:

````md
```ts
export function formatTitle(title: string) {
  return title.trim().replace(/\s+/g, ' ');
}
```
````

That will render as:

```ts
export function formatTitle(title: string) {
  return title.trim().replace(/\s+/g, ' ');
}
```

## Common languages

Compass will highlight many common languages out of the box, including:

- `ts`
- `js`
- `tsx`
- `jsx`
- `json`
- `bash`
- `html`
- `css`
- `astro`
- `md`

## Theme configuration

Compass uses Astro's built-in Markdown highlighting through `shikiConfig` in `astro.config.mjs`:

```js
markdown: {
  shikiConfig: {
    theme: 'github-light',
  },
},
```

If you want to use a different built-in Shiki theme later, update that value and rebuild the site.

## When to use inline code
