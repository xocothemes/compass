Analytics can help you decide which articles deserve more depth, which pages go stale, and where readers drop off.

## Useful signals for docs

- page views by article
- top landing pages from search
- exit-heavy pages that may need clearer next steps
- internal search queries that reveal missing docs

## Good fits for Compass

Compass can work with lightweight or product analytics tools such as:

- Plausible
- Fathom
- PostHog
- Google Analytics

## Where to wire it in

Most teams add the tracking script in the shared layout so it loads across the docs site.

## Start small

If you do not need event-heavy product analytics, a privacy-friendly page analytics tool is usually enough for a docs site.
