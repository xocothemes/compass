Embedded help panels bring documentation closer to the moment a user needs it.

## When to use an embed

- On settings pages with complex options
- Next to onboarding steps
- Inside admin flows that need more explanation

## Example embed container

```html
<aside class="help-panel">
  <h2>Need help?</h2>
  <p>Open the setup guide without leaving this page.</p>
</aside>
```
